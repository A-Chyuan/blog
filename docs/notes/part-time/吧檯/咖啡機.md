# 咖啡機

## 午班

用清潔錠與清水清洗牛奶管（按 $151511$）。

## 晚班

|     步驟      |                         細節                          |
| :-----------: | :---------------------------------------------------: |
|   咖啡渣槽    |                      沖洗並晾乾                       |
|   拆除零件    |        長管<br />短管<br />奶管零件 $\times 3$        |
|   浸泡零件    |     水 $1L$<br />藥水 $50cc$<br />浸泡 $15$ 分鐘      |
|   刷洗零件    |                      刷除牛奶垢                       |
|   安裝零件    |                     短管黑上紅下                      |
|     清洗      | 長按按鍵 $0$<br />按 $151311$<br />選項「咖啡與牛奶」 |
|  $0\%～5\%$   |              加入清潔錠<br />按 $0$ 清洗              |
|  $5\%～50\%$  |           將奶管放入藥水中<br />按 $0$ 清洗           |
| $50\%～100\%$ |            將藥水換成清水<br />按 $0$ 清洗            |
|  回初始頁面   |          按五次 $0$<br />顯示「請選擇飲料」           |
|     關機      |                                                       |
