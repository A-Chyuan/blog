# 爆香

## 爆香料

|          品項          |       爆香料       |
| :--------------------: | :----------------: |
|         清炒類         |        蒜片        |
|       紅醬、青醬       |     洋蔥、蒜片     |
|          白醬          |    洋蔥、紅蔥頭    |
|   明太子、墨汁、淡菜   | 洋蔥、蒜片、紅蔥頭 |
| 茄菇、菜麵<br />野菇類 |         無         |
